# 📦 SistemaControlDeStock

## 📌 Descripción

Sistema de Control de Stock, diseñado para facilitar la administración de productos, inventario y movimientos de stock de manera eficiente y segura.

El sistema permite gestionar entradas y salidas de productos, mantener actualizado el inventario en tiempo real y obtener un mayor control sobre las operaciones comerciales.

---

## 🚀 Funcionalidades principales

* 📦 Gestión de productos.
* 🏷️ Registro de categorías.
* 📥 Registro de ingresos de mercadería.
* 📤 Registro de egresos y ventas de productos.
* 📊 Control y actualización automática del stock.
* 💰 Gestión de precios de compra y venta.
* 👥 Administración de clientes y proveedores.
* 🔍 Búsquedas y filtros de información.
* 📈 Reportes de movimientos de inventario.
* 🔐 Sistema de autenticación y seguridad de usuarios.

## ⚙️ Instalación

### Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

### Ejecutar Backend

```bash
mvn spring-boot:run
```

El servidor estará disponible en:

```
http://localhost:8085
```

### Ejecutar Frontend

```bash
npm install
npm run dev
```

El sistema estará disponible en:

```
http://localhost:5173
```

---

## 🗂️ Estructura del sistema

El proyecto está dividido en dos partes principales:

* **Backend:** API REST encargada de la lógica del negocio, seguridad y acceso a datos.
* **Frontend:** Interfaz web para la interacción del usuario con el sistema.
